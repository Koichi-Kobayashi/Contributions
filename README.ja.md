# GitHub Contribution Chart Generator – Desktop Edition

GitHub Contribution Chart Generator – Desktop Edition は、GitHub のコントリビューション履歴を可視化するシンプルなデスクトップアプリケーションです。

本アプリは、以下の Web アプリにインスパイアされて作成されています。

- https://github.com/sallar/github-contributions-chart

---

## 概要

指定した **GitHub アカウントの Profile ページ** を読み込み、  
**Contributions（コントリビューション）** を解析して、  
その結果をデスクトップアプリ上に表示します。

ブラウザを使用せずに、手軽に GitHub の活動状況を確認することができます。

---

## 動作概要

1. 指定した GitHub アカウントの Profile ページを読み込み  
2. Profile ページ内の Contributions 情報を解析  
3. コントリビューション履歴を集計  
4. デスクトップアプリ上にコントリビューションチャートを表示  

---

## 実行方法

1. 本リポジトリを pull（clone）します  
2. **Visual Studio 2026 以上**でソリューションを開きます  
3. **F5** キーを押して実行します  

特別な設定や追加作業は不要です。

---

## 動作環境

- Windows
- Visual Studio 2026 以上
- インターネット接続（GitHub の Profile ページ取得に使用）
